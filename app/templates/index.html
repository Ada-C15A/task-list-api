Available Routes

PATH METHOD REQUEST BODY DESCRIPTION
/tasks GET N/A Get all tasks. Optional parameter 'sort=<asc or desc>' to sort by task title.
/tasks POST 'title', 'description', 'completed_at' Create a task. 'completed_at' can be null.
/tasks/<task_id> GET N/A Get details for a single task.
/tasks/<task_id> PUT 'title', 'description', 'completed_at' Update details for a single task.
/tasks/<task_id> DELETE N/A Delete a single task.
/tasks/<task_id>/mark_complete PATCH N/A Set 'completed_at' for the given task to the current time.
/tasks/<task_id>/mark_incomplete PATCH N/A Set 'completed_at' for the given task to null.
/goals GET N/A Get a list of all goals.
/goals POST "title" Create a goal.
/goals/<goal_id> GET N/A Get deails for a single goal.
/goals/<goal_id> PUT 'title' Update details for a single goal.
/goals/<goal_id> DELETE N/A Delete a single goal.
/goals/<goal_id>/tasks GET N/A Get all the tasks for a single goal.
/goals/<goal_id>/tasks POST 'task_ids':['task_id'] Set the goal_id to for each task in the request body.
